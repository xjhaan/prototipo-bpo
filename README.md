# Prototipo BPO - Sistema de Gestión

Prototipo de baja fidelidad para un sistema de gestión de Business Process Outsourcing (BPO).

## Características

- ✅ Login con roles diferenciados (Cliente/BPO)
- ✅ Búsqueda de BPOs con filtros (tipo de servicio, precio)
- ✅ Creación de proyectos y carga de documentos
- ✅ Validación y aprobación de proyectos
- ✅ Dashboard específico para Cliente y BPO
- ✅ Historial de proyectos

## Tecnologías

- React
- Tailwind CSS
- Lucide React (iconos)

## Cómo ejecutar
```bash
npm install
npm start
```

La aplicación abrirá en `http://localhost:3000`

## Flujos Principales

### Cliente
1. Login como Cliente
2. Buscar BPOs disponibles
3. Crear nuevo proyecto
4. Cargar documentos
5. Ver estado y aprobar/rechazar resultados

### BPO
1. Login como BPO
2. Ver proyectos asignados
3. Descargar documentos
4. Subir resultado
5. Esperar aprobación

## Pantallas

- **Login** - Selección de rol
- **Dashboard Cliente** - Búsqueda de BPOs
- **Crear Proyecto** - Formulario
- **Cargar Documentos** - Drag-and-drop
- **Estado del Proyecto** - Validación
- **Dashboard BPO** - Proyectos asignados
